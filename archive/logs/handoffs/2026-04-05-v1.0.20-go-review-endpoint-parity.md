# Handoff Archive — v1.0.20 Go Review Endpoint Parity

## Summary
This pass ported Go-native review endpoints so direct code-review workflows can run through the Go backend instead of depending on Bun-only review routing.

## Shipped
- Added `backend-go/services/review.go`.
- Added `POST /api/review` to the Go API.
- Added `POST /api/local/review` to the Go API.
- Added simple, comprehensive, and structured JSON review modes in Go.
- Reused the Go provider bridge for review execution.
- Bumped version surfaces to `1.0.20`.

## Why This Matters
Review is an actively used UI workflow. Porting it removes another practical Bun-only dependency and expands the Go backend's product-surface credibility beyond the core autonomy loop.
